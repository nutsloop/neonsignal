# NeonSignal Synthwave (Zed Theme)

A synthwave-inspired theme pack for the Zed editor, tuned to match the NeonSignal aesthetic.

## Install (Manual)

1. Copy this directory to your Zed extensions folder:
   - macOS: `~/.config/zed/extensions/neonsignal-synthwave/`
   - Linux: `~/.config/zed/extensions/neonsignal-synthwave/`

2. Ensure the theme JSON is present:
   - `themes/synthwave.json`

3. Restart Zed.

4. Select the theme:
   - `Settings → Themes → Synthwave Dark` (or `Synthwave Light`).

## Files

- `extension.toml` — Zed extension manifest
- `themes/synthwave.json` — Theme definition

## Source of Truth

The canonical theme source lives here:
- `tools/zed-theme/synthwave.json`

If you update the source, re-copy it into the extension directory.
